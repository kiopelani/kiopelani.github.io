JavaScript: Pros and Cons
August 23, 2014

JavaScript is one of the most popular languages out there now, and some predict it's well on its way to becoming the dominant programming language of the not so distant future. So what makes JavaScript so beloved, and are there any drawbacks to this language?

Pros:

  1. JavaScript syntax resembles English, meaning it is easy to learn and easy to maintain
  2. JavaScript plays well with others, meaning it can be used with other languages to enhance web pages and applications
  3. JavaScript saves time and bandwidth, because it is executed on the client side, meaning functions run immediately on the user's computer rather than waiting for approval from the server

Cons:

  1. Security Problems: Because JavaScript executes immediately on the client side, this means that malicious applications can be easily executed, which puts the client at risk
  2. Variety of Interpretations: Different browsers have different ways of interpreting JavaScript, meaning the results can vary
