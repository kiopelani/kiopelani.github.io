Reflection

Well, this was my first pairing, so one of the biggest challenges was just figuring out the logistics of how a remote pairing session works. Once we figured it out though, the process was pretty smooth (and I learned a lot!).

One of the main problems we had was figuring out how to get the unordered list items in the header to appear horizontally on the screen. At first we tried just doing "display: inline", but eventually we realized that the width of our column (15.625%) wasn't big enough to allow our text to extend across the page, so we changed it to 100%.

As for general concepts, I'm still a little fuzzy on when we should be using Media Queries vs making the original layout liquid (we didn't end up using Media Queries on this, as our page seemed to adapt to smaller screens well without them).
